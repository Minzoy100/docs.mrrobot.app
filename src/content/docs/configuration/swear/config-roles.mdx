Vous pouvez **configurer les rôles** qui ne seront pas concernés par le filtre contre les insultes.

```
/paramètres-insultes Paramétrer les rôles
```

![Mr. Robøt : Paramétrer les rôles](https://i.thomasbnt.dev/i/ng087.png)

:::note
Il n'y a aucun rôle configuré par défaut, ce qui signifie que le filtre contre les insultes s'appliquera sur tous les rôles de votre serveur **sauf les modérateurs**.
:::

## Vous avez la permission de gérer les messages

Si vous avez la permission de gérer les messages, soit de supprimer un message sur le serveur Discord en question, vous n'êtes donc **pas affecté** par le filtre.
