## Configurer le filtre

Vous avez la possibilité **d'activer et de désactiver** le filtre contre les insultes sur **l'entièreté de votre serveur** avec la commande adéquat.

```
/paramètres-insultes Activer/désactiver le filtre contre les insultes ⚠
```

Même si le filtre est désactivé, vous pouvez toujours configurer les salons qui ne seront pas concernés par celui-ci, ainsi que les rôles.

:::note
Le filtre contre les insultes est activé par défaut.
:::

## Configurer les salons

Vous pouvez **configurer les salons** qui ne seront pas concernés par le filtre contre les insultes.

```
/paramètres-insultes Paramétrer les salons
```

![Mr. Robøt : Paramétrer les rôles](https://i.thomasbnt.dev/i/453in.png)

:::note
Il n'y a aucun salon configuré par défaut, ce qui signifie que le filtre contre les insultes s'appliquera sur tous les salons de votre serveur.
:::

## Configurer les rôles

Vous pouvez **configurer les rôles** qui ne seront pas concernés par le filtre contre les insultes.

```
/paramètres-insultes Paramétrer les rôles
```

![Mr. Robøt : Paramétrer les rôles](https://i.thomasbnt.dev/i/ng087.png)

:::note
Il n'y a aucun rôle configuré par défaut, ce qui signifie que le filtre contre les insultes s'appliquera sur tous les rôles de votre serveur **sauf les modérateurs**.
:::
