<Aside type="danger" title="Changement !">
  Auparavant, les modérateurs étaient exclus du filtre, ce n'est plus le cas. Vous devez désormais les ajouter manuellement.
</Aside>

Vous pouvez **configurer les rôles** qui ne seront pas concernés par le filtre contre les insultes.

```
/paramètres-insultes Paramétrer les rôles
```

![Mr. Robøt : Paramétrer les rôles](https://i.thomasbnt.dev/i/ng087.png)

<Aside type="note" title="Par défaut">
  Il n'y a aucun rôle configuré par défaut, ce qui signifie que le filtre contre les insultes s'appliquera sur tous les rôles de votre serveur.
</Aside>
