Vous pouvez **configurer les salons** qui ne seront pas concernés par le filtre contre les insultes.

```
/paramètres-insultes Paramétrer les salons
```

![Mr. Robøt : Paramétrer les rôles](/img/swear/configure_channels_swear_system.png)

:::note
Il n'y a aucun salon configuré par défaut, ce qui signifie que le filtre contre les insultes s'appliquera sur tous les salons de votre serveur.
:::

## Permission

Permission par défaut : **Gérer le serveur**.
