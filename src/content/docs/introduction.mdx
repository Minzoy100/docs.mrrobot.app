Cette documentation est un guide d'introduction à l'utilisation de Mr. Robøt. Elle est destinée à vous aider sur les différentes commandes et fonctionnalités du robot.

## C'est quoi Mr. Robøt ?

[Mr. Robøt](https://mrrobot.app) est un **robot Discord** qui permet de gérer votre serveur Discord. Il permet de supprimer par défaut les insultes venant de tout membre tout en étant configurable.

## Comment utiliser cette documentation ?

Cette documentation est organisée en plusieurs guides, chacun couvrant un sujet particulier. Vous pouvez lire les guides dans l'ordre, ou simplement aller directement à un sujet qui vous intéresse.

## Contribuer à cette documentation

Cette documentation est hébergée sur GitHub. Si vous souhaitez contribuer à cette documentation, vous pouvez cliquer sur le bouton "Modifier cette page" en bas de chaque page et soumettre une demande de tirage (pull request).
Toute contribution est la bienvenue, qu'il s'agisse de corriger une faute de frappe ou d'ajouter un guide complet.
