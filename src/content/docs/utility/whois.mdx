La commande `/whois` permet de voir les informations d'un utilisateur Discord.

![Mr. Robøt : Fonctionnalité primaire, des commandes utilitaires sur Discord](https://mrrobot.app/img/main_features/utilitaire.png)

## Utilisation

Pour voir les informations d'un utilisateur, vous devez utiliser la commande `/whois` suivie de l'utilisateur ciblé.

```
/whois @utilisateur
```

![Utilisation de la commande /whois](https://i.thomasbnt.dev/i/qqs03.png)

Pour voir **vos informations**, vous pouvez utiliser la commande `/whois` sans argument.

```
/whois
```

## Menu d'applications

Vous pouvez également utiliser la commande `/whois` en faisant un clic droit sur un utilisateur.

*Clic droit sur un utilisateur > `Applications` > `Plus sur son profil`.*

![Utilisation de la commande /whois avec un clic droit](https://i.thomasbnt.dev/i/4k1ke.png)
