The `/clear` command is used to delete a certain number of messages in a channel.
This command is reserved for server moderators.

## Usage

```
/clear <number>
```

:::note
The number of messages to be deleted must be between 2 and 100. The limit is imposed by Discord.
:::

## Permission

Default permission: **Manage messages**.
